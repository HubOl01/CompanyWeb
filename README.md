# CompanyWeb

**Стек технологий:**
- Бэкенд: C#, ASP .Net Core
- Фронтенд: Angular
- База данных: MS SQL(*.mdf)
- Верстка: Bootstrap

**Среда разработки:** Visual studio 2022

**Менеджер БД:** SSMS


## Angular

Установка необходимых библиотек для начала
```bash
npm install
```

Запуск приложения в консоли
```bash
ng serve -o
```
